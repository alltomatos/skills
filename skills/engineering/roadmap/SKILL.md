# Roadmap

Esta skill gerencia o arquivo `ORCHESTRATOR-ROADMAP.md` na raiz do repositório, que serve como a bússola estratégica para o projeto.

## Quando usar

- Ao iniciar um projeto para definir o plano macro.
- Antes de cada Milestone para detalhar as tarefas.
- Para verificar o progresso global.

## Estrutura do Roadmap

O arquivo deve conter:
1. **Epics**: Grandes objetivos do projeto.
2. **Milestones**: Marcos entregáveis compostos por tarefas.
3. **Estado**: `todo`, `in_progress`, `done`.

## Fluxo

1. **Leitura**: Verifica o estado atual do `ORCHESTRATOR-ROADMAP.md`.
2. **Atualização**: Adiciona ou modifica Epics/Milestones com base no progresso.
3. **Persistência**: Garante que o roadmap esteja sempre refletindo a realidade do código.
